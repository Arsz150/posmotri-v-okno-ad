# 👀 Посмотри в окно

Одностраничное приложение для поиска городских веб-камер по городу и времени суток.

## О проекте

«Посмотри в окно» позволяет найти видеотрансляции из разных городов и посмотреть, что происходит за окном прямо сейчас.

Пользователь может:

* указать название города;
* выбрать время суток;
* получить список подходящих трансляций;
* просмотреть найденное видео.

Проект выполнен в рамках обучения на курсе **Frontend-разработчик** от Яндекс Практикума.

## Технологии

* HTML5
* CSS3
* JavaScript (ES6)
* БЭМ
* Git

## Функциональность

* Поиск по названию города
* Фильтрация по времени суток
* Динамическая подгрузка результатов
* Отображение выбранного видео
* Состояния элементов интерфейса:

  * hover
  * focus-visible
  * active
  * disabled

## Установка и запуск

Клонируйте репозиторий:

```bash
git clone https://github.com/Arsz150/posmotri-v-okno-ad.git
```

Перейдите в папку проекта:

```bash
cd posmotri-v-okno
```

Откройте файл `index.html` в браузере.
